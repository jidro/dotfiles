# Mine Conky Config

## 中文说明：

该文件夹为之前或正在使用的`Conky`的配置文件。

### 文件说明：

- [`mine-conky.conf.brillant`](https://github.com/jidro/conky/blob/master/mine-conky/mine-conky.conf.brillant) - 为之前使用的`Conky`的配置文件，该配置文件集合了所有可能用到的参数。

- [`mine-conky.conf.minimalist`](https://github.com/jidro/conky/blob/master/mine-conky/mine-conky.conf.minimalist) - 为正在使用的`Conky`的配置文件，在[`mine-conky.conf.brillant`](https://github.com/jidro/conky/blob/master/mine-conky/mine-conky.conf.brillant)的基础之上进行精简而来。

- [`mine-conky.conf.simple`](https://github.com/jidro/conky/blob/master/mine-conky/mine-conky.conf.simple) - 为之前使用的`Conky`的配置文件，该配置文件是根据官方文档一步步写出来的。

- [`mine-conky.conf.time`](https://github.com/jidro/conky/blob/master/mine-conky/mine-conky.conf.time) - 为正在使用的`Conky`的配置文件，该配置文件以时间为主体，仅有极少的配置。

## English description：

This folder is the configuration file for `Conky` that has been used or is being used.

### Filespec：

- [`mine-conky.conf.brillant`](https://github.com/jidro/conky/blob/master/mine-conky/mine-conky.conf.brillant) - Is the configuration file for `Conky` that was used earlier. This configuration file is a collection of all possible parameters.

- [`mine-conky.conf.minimalist`](https://github.com/jidro/conky/blob/master/mine-conky/mine-conky.conf.minimalist) - For the `Conky` configuration file being used, it is simplified based on [`mine-conky.conf.brillant`](https://github.com/jidro/conky/blob/master/mine-conky/mine-conky.conf.brillant).

- [`mine-conky.conf.simple`](https://github.com/jidro/conky/blob/master/mine-conky/mine-conky.conf.simple) - The configuration file for `Conky` is previously used. The configuration file is written step by step according to the official documentation.

- [`mine-conky.conf.time`](https://github.com/jidro/conky/blob/master/mine-conky/mine-conky.conf.time) - Is the configuration file for the `Conky` that is being used. This configuration file is time-oriented and has very few configurations.
