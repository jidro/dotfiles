# `ATerm`

------

`aterm` - 具有透明支持的`xterm`替代。    </br>

------

## 中文说明：

该仓库为本人所使用的`aterm`终端配置文件。    </br>

------

### 使用说明：

- ① 使用如下命令，将该仓库中的配置文件用户主目录（`~/`）下：    </br>
  
  ```shell
  $ cp -rfpv .Xdefaults ~/
  ```

- ② 重启`aterm`终端，以使用修改好的配置文件。    </br>

> `aterm`配置文件`.Xdefaults`(隐藏文件)导出路径：`~/`
